Font: Bear

Created: 2021

BEAR was born as a font for a logo concept. This was developed after using a bunch of letter for a logo (that was never used).

This typeface is condensed. It looks like it's stretched, like if someone stepped on the entire alphabet.

It has something in common with the SPACE font that you have seen before. The shapes are almost the same but with sharp corners.

This font is perfect for big titles and data representation, and, again, not very good for paragraphs.
